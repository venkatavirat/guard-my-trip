# Guard My Trip

Guard My Trip is a frontend prototype of a travel safety application designed to help users stay safe while exploring new places. The app focuses on proactive safety by showing alerts, highlighting risky areas, and providing an instant emergency response feature.

---

## 🌍 Project Overview

Travelers often lack access to reliable, real-time safety information. Guard My Trip addresses this by simulating:

- 🚨 Safety alerts based on location
- 🗺️ Risk zone visualization (Red / Yellow / Green)
- 🆘 One-tap SOS emergency feature

This project demonstrates how a safety-focused product can be structured and experienced at the UI level.

---

## ✨ Features

- **Safety Alerts**
  - Displays risk levels for different areas
  - Simulates real-time alert system

- **Risk Zones**
  - Categorizes areas into:
    - 🔴 High Risk
    - 🟡 Medium Risk
    - 🟢 Safe

- **Emergency SOS**
  - One-click emergency trigger
  - Simulated response confirmation
